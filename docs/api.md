| **项目**      | **内容**                                                     |
| ------------- | ------------------------------------------------------------ |
| **命令名称:** | **SendCommand 发送命令**                                     |
| **命令功能:** | 在 Controller 或 System 中执行具体的业务逻辑命令，实现逻辑封装。 |
| **语法格式:** | `this.SendCommand<T>(new T());` 或 `this.SendCommand(new T(params));` |
| **参数说明:** | **T:** 继承自 `AbstractCommand` 的类。 **params:** 命令构造函数所需的参数。 |
| **返回值:**   | **void:** 无返回值（或根据具体实现返回 Command 自身）。      |

| 项目                                  | 内容                                                         |
| :------------------------------------ | :----------------------------------------------------------- |
| **命令名称**                          | **GetNetworkTime 获取网络时间**                              |
| **命令功能**                          | 获取当前的网络时间                                           |
| **语法格式**                          | `GetNetworkTime()`                                           |
| **参数说明**                          | 无                                                           |
| **返回值**                            | 返回当前的网络时间，例如：2015-09-01 10:54:49                |
| <font color="red">**脚本例子**</font> | `Dim 网络时间`<br>`网络时间 = GetNetworkTime()`<br>`Traceprint 网络时间` |
| **备注**                              | 无                                                           |
